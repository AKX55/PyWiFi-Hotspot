# PyWiFi-Hotspot
Wifi Hotspot Creator script for Windows

**Setting Up hotspot:**
-Open Cmd As Administrator
-Use the following command to start Hotspot: 
	

    python set-wifiHotspot.py start "WiFi" "12345678"
-To Stop Hotspot:

    python set-wifiHotspot.py stop
